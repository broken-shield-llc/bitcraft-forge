# Contributing

This repository is **public** so anyone can **read**, **fork**, and **use** the software under the [LICENSE](LICENSE) (MIT).

**Issues are welcome** for bug reports and feature ideas. There is no guarantee of a response or timeline; this is maintained in spare time.

**Pull requests are not accepted.** If you have a concrete change in mind, open an issue to discuss first, or maintain a **fork** with your own patches.

If you find a **security problem**, follow [SECURITY.md](SECURITY.md) instead of filing a public issue with exploit details.
